Flujo normal del caso de uso<br/>
Steps:
1. El anunciante selecciona " Imprimir informes "<br/>
2. El sistema muestra todo el contenido publicitario proporcionado por el anunciante<br/>
3. El anunciante selecciona una o más piezas de contenido sobre las que informar<br/>
4. El sistema muestra una lista de informes para este anunciante<br/>
5. El anunciante selecciona uno o más informes para generar<br/>
6. El anunciante selecciona el formato (Microsoft® Word®, Microsoft® Excel® o la ventana del navegador)<br/>
7. El sistema crea el primer informe y le pide al usuario que lo guarde o lo vea<br/>
8. El anunciante guarda o ve el informe, selecciona "Informe siguiente"<br/>
9. El sistema crea el siguiente informe y le pide al usuario que lo guarde o lo vea<br/>
10. Las viñetas 8 y 9 se repiten hasta que todos los informes estén completos<br/>
