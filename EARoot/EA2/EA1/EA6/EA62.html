Details:
Esto debe asegurarse, ya que la información de la tarjeta de crédito puede estar en el perfil<br/>
Scenario Type
Flujo básico Basic Path
Details:
Flujo normal para el caso de uso<br/>
Steps:
1. El usuario selecciona " Editar perfil "<br/>
2. El sistema muestra categorías de perfil (personal, preferencias, información del buscapersonas, selecciones de "enviarme un mensaje cuando").<br/>
3. El usuario selecciona la categoría<br/>
4. El sistema muestra el detalle<br/>
5. El usuario actualiza el detalle, presiona "Aceptar"<br/>
6. El sistema valida los datos según sea necesario, actualiza el perfil del suscriptor.<br/>
Suscriptor nuevo Alternate
Details:
Si se trata de un suscriptor nuevo, se invoca el caso de uso "Pago de tarifa con tarjeta de crédito" siguiendo el paso 5 anterior.<br/><br/>
