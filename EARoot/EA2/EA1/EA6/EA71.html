Scenario Type
Flujo básico Basic Path
Details:
Flujo normal de la aplicación<br/>
Steps:
1. El sistema analiza la lista de contenido "archivado". Para cualquier historia de más de 2 días, la historia se mueve de nuevo a la categoría general.<br/>
2. El sistema muestra anuncios publicitarios, categorías de contenido general e historias específicas para las que se han enviado páginas.<br/>
3. Suscriptor visualiza historias<br/>
4. Para cualquier historia paginada, las historias se marcan como vistas y se colocan en una categoría "archivada".<br/>
El usuario no es un suscriptor registrado Alternate
Details:
El usuario no ha pagado la suscripción<br/>
Steps:
1. El sistema muestra anuncios publicitarios y categorías de contenido general<br/>
2. El sistema brinda la opción para que el usuario se suscriba<br/>
3. Historias de visitas de usuarios<br/>
