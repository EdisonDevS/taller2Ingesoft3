Scenario Type
Flujo básico Basic Path
Details:
Flujo normal del caso de uso<br/>
Steps:
1. El sistema verifica las categorías para el nuevo contenido<br/>
2. El sistema verifica las listas de suscriptores para determinar si algún suscriptor desea ser localizado para esta categoría de contenido<br/>
3. El sistema genera un mensaje de texto basado en el título<br/>
4. El sistema construye una serie de mensajes de correo electrónico<br/>
5. El sistema envía mensajes de correo electrónico a los suscriptores (quienes los recibirán como una página alfanumérica)<br/>
