Este caso de uso ocurre cuando un usuario del sistema (anunciante, suscriptor o suscriptor potencial) desea comentar sobre el servicio o el sitio web<br/>
  • Scenarios
  • Advanced
  • Other Links
Scenario Type
Flujo básico Basic Path
Details:
Flujo normal de la aplicación<br/>
Steps:
1. El usuario selecciona " Proporcionar comentarios "<br/>
2. El sistema busca los números de teléfono de soporte del usuario central<br/>
3. El sistema muestra los números de teléfono para llamar y le da al usuario la opción de enviar un correo electrónico inmediatamente<br/>
4. El usuario selecciona la opción de correo electrónico<br/>
5. El sistema busca la dirección de correo electrónico del servicio al cliente y la pasa al navegador<br/>
6. El sistema inicia el cliente del navegador de correo electrónico<br/>
7. El usuario ingresa el mensaje, presiona "Enviar"<br/>
8. El cliente de correo del navegador envía correo<br/>
